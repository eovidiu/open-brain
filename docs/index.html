Every AI forgets you

Switch from Claude to ChatGPT to Cursor — and start over every time. Your context, decisions, and insights vanish between sessions and tools.

THE FIX
One brain, every AI

Open Brain gives all your AI tools shared memory. Capture a thought in Claude, recall it from Cursor. Your data, your server, always available.

Up and running in 3 steps

No coding required. The setup wizard handles everything.

01

Create accounts

Sign up for Supabase (free tier) and get an OpenAI API key for embeddings.

02

Run the wizard

One command creates your database, deploys the server, and sets all secrets.

npx @eovidiu/open-brain-setup
03

Connect your AI

Paste the config into Claude Desktop, Cursor, or any MCP client. Done.
